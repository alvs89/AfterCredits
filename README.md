# AfterCredits

AfterCredits is a comprehensive, privacy-first personal media tracking application. Built as an offline-capable Progressive Web App using React and Dexie.js (IndexedDB), it allows users to effortlessly log, rate, and review their favorite Movies, TV Series, K-Dramas, and Anime. It features a rich text journal for episodic notes, custom platform tagging, statistical dashboards, and an elegant dark/light mode interface—all securely stored directly on the user's device.
